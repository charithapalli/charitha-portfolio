# Charitha Palli — Portfolio

This is a simple, fast, single-page portfolio built for GitHub Pages.

## Quick start (local)
Open `index.html` in your browser, or use any local web server:
- VS Code: **Live Server**
- Python:
  - `python -m http.server 8000` then open `http://localhost:8000`

## Customize
Edit these in `index.html`:
- LinkedIn + GitHub URLs
- Project links (GitHub/Demo)
- Resume file name (stored in `assets/`)

## Deploy to GitHub Pages
1. Create a new GitHub repo (e.g., `charitha-portfolio`)
2. Upload the files from this folder (or push via git)
3. GitHub → **Settings** → **Pages**
4. Source: **Deploy from a branch**
5. Branch: `main` and folder: `/ (root)`
6. Wait for the site to publish, then open the URL GitHub provides.

Optional: set a custom domain in the Pages settings.
